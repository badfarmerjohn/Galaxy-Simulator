CS 184: Computer Graphics and Imaging, Spring 2019

Final Project Milestone: Galaxy Simulator

Jeromy Lui, Ryan Meyer, Eric Zhang



Status Report

We have accomplished the foundational behavior for the camera in Unity, CPU-based particles (which will make up the physically simulated particles in our galaxy), and GPU particles (which will be used for realistic visual effects which are not part of the physical simulation). The CPU particles will have their positions updated by computing the dynamics of an N-body gravitational system. The GPU particles will be used to mimic real phenomena such as space dust which are present in the simulation but do not affect the dynamics of the N-body system. The progress relative to our plans is that we have yet to make the galaxy very visually stunning or compelling. And as of now, we are only using one mathematical model to model our galaxies. We have plans to implement more models of more variety. In terms of efficieny, our current code seems to be on the right track. The GPU-based code for visual effects graph is much more efficient than the CPU-based code for simulating the particles in the galaxy.
