Status Report

We have accomplished a basic galaxy simulator using unity. As of right now, the galaxy model is based off the gaussian distribution on the x and z axis. For the y axis, we uniformly sample in the range of the guassian distribution. The preliminary results are that ... The progress relative to our plans is that we have yet to make the galaxy very visually stunning or compelling. And as of now, we are only using one mathematical model to model our galaxies. We have plans to implement more models of more variety. In terms of efficieny, our current code seems to be on the right track. The GPU-based code for visual effects graph is much more efficient than the CPU-based code for simulating the particles in the galaxy.
